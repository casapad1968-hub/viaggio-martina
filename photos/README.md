# 📸 Foto del viaggio

Questa cartella è l'**archivio locale** delle foto del viaggio di Martina & Riccardo.

## ⚠️ Privacy

**Le foto NON vengono pubblicate nel sito web pubblico** (https://casapad1968-hub.github.io/viaggio-martina/) — sono solo archivio personale.

Il sito pubblico usa esclusivamente le illustrazioni SVG manga presenti in `app.js`.

## 📁 Convenzione nomi file

Quando aggiungi foto, usa nomi descrittivi per ritrovarle facilmente:

```
photos/
├── 2026-07-18_milano-malpensa_partenza.jpg
├── 2026-07-19_bangkok_mandarin-hotel_room.jpg
├── 2026-07-20_bangkok_grand-palace.jpg
├── 2026-07-23_okinawa_naha-airport.jpg
├── 2026-07-28_naoshima_yam-art.jpg
├── 2026-08-02_seoul-myeongdong_street-food.jpg
└── ...
```

Formato: `YYYY-MM-DD_<luogo>_<soggetto>.jpg`

## 🔒 Consigli

- Prima di committare foto personali, **verifica cosa stai pushando** (potrebbero esserci volti, documenti, ecc.)
- Considera l'uso di `.gitignore` per escludere le foto dal repo pubblico
